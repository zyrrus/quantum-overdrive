Quantum Overdrive Release Date:

Updates:

Started work on Quantum Overdrive!

2/21/22

Quantum Dev is excited to announce that we have started work on a new, sci-fi/dystopian, open-world adventure.
Concept art coming soon!

Game Functions and art work have been created!

3/28/22

The current updates include that Character model + animations are nearly finalized, Movement system has nearly been fully implemented, Some non-vital actions still need to be implemented, Some actions (such as attacking) are now influenced by movement speed, and a simplified version of the combat system has been implemented Currently, there is no model/animation for the attack, so we’re temporarily drawing attacks with Unity’s Gizmos instead. A stationary turret enemy has been created along with a system for enemies to detect a nearby player

New Art Direction!

4/27/22

The current update includes the transition from 3D to 2D. We have adding in more art to give a much better aestheic appeal.

Final Release Post!

5/2/22

The Game is completed and ready to be played! With the end of this semester comes the end of our project journey. With this final product, we managed to achieve several of the goals set out in our initial project pitch. We created art assets for our main character, initial levels have been designed, new systems have been created, and basic enemy templates have been created.
